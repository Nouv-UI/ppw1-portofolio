tag preformat menampilkan teks sesuai format yang kita tuliskan di code editor

        sebagai contoh teks ini yang     spasi dan menggunakan line break
        tanpa menggunakan    tag br

tag ini cocok digunakan untuk menampilkan source code. Misal:

    for i in range(len(nama)):
        avg = (matkul1[i] + matkul2[i]) / 2
        print(i+1:3 | nama[i]:10 | matkul1[i] | matkul2[i] | avg)
        print("-" * 50)

karena tag pre menampilkan teks sesuai format di code editor, jadi font nya juga menyesuaikan dengan yang dipakai di code editor
